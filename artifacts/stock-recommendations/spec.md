# spec.md — 한국 주식 전문가 추천종목 요약 서비스

## 개요

여러 증권사에 흩어진 전문가 추천종목 정보를 한 화면에 모아, 사용자가 매일 아침 30초 안에 오늘의 추천종목과 컨센서스를 파악할 수 있게 한다. 추천종목 데이터는 주기적으로 백그라운드에서 갱신되고, 현재가·상승여력은 실시간으로 표시된다.

## 범위

### 포함

- 오늘의 추천종목 목록 (종목명, 증권사, 투자의견, 목표가, 현재가, 상승여력)
- 종목 클릭 시 상세 모달 (컨센서스 요약, 목표가 히스토리 차트, 증권사별 리포트 리스트)
- 트렌드 랭킹 섹션 (최근 N일간 가장 많이 추천된 종목 Top 10, 기간: 5일/20일/60일 선택)
- 현재가·상승여력 실시간 표시
- 추천종목·컨센서스 데이터 주기적 백그라운드 갱신

### 제외

- 로그인·사용자 계정 (MVP에서 불필요, 개인화 기능 없음)
- 매매 기능 및 포트폴리오 관리 (서비스 범위 밖)
- 리포트 원문 제공 (저작권 이슈)
- 알림·푸시 기능 (백엔드 인프라 필요, MVP 이후)
- 종목 검색·필터·정렬 (MVP 이후 추가)

## 시나리오

### 1. 오늘의 추천종목 목록 확인

- **Given** — 사용자가 서비스 홈페이지에 접속한다
- **When** — 페이지가 로드된다
- **Then** — 오늘 날짜 기준 추천종목 목록이 표시된다. 각 행은 종목명, 증권사, 투자의견, 목표가, 현재가, 상승여력을 포함한다

성공 기준:
- [ ] 페이지 진입 시 "오늘의 추천종목" 제목과 함께 종목 행이 1개 이상 렌더된다
- [ ] 각 행에 종목명, 증권사명, 투자의견(예: 매수), 목표가(원 단위), 현재가(원 단위), 상승여력(% 표기)이 표시된다
- [ ] 현재가와 상승여력은 실시간 시세 기준으로 표시된다 (정적 값이 아님)

### 2. 현재가·상승여력 실시간 갱신

- **Given** — 추천종목 목록이 화면에 표시된 상태
- **When** — 실시간 시세가 업데이트된다
- **Then** — 사용자 조작 없이 현재가와 상승여력이 갱신된 값으로 변경된다

성공 기준:
- [ ] 현재가가 변경되면 해당 행의 현재가 셀이 새 값으로 업데이트된다
- [ ] 현재가 변경에 따라 상승여력(%)이 재계산되어 표시된다

### 3. 종목 상세 모달 열기

- **Given** — 추천종목 목록이 화면에 표시된 상태
- **When** — 사용자가 특정 종목 행을 클릭한다
- **Then** — 해당 종목의 상세 모달이 열린다. 모달에는 컨센서스 요약, 목표가 히스토리 차트, 증권사별 리포트 리스트가 포함된다

성공 기준:
- [ ] 종목 행 클릭 시 모달이 열린다
- [ ] 모달 상단에 컨센서스 요약(평균 목표가, 의견 분포: 매수/중립/매도 건수)이 표시된다
- [ ] 모달 내 차트 영역에 날짜별 목표가 히스토리가 표시된다
- [ ] 모달 내 리스트에 증권사별로 날짜, 투자의견, 목표가가 나열된다
- [ ] 모달 외부 클릭 또는 닫기 버튼 클릭 시 모달이 닫힌다

### 4. 트렌드 랭킹 확인

- **Given** — 사용자가 홈페이지에 접속한 상태
- **When** — 트렌드 랭킹 섹션을 본다
- **Then** — 기본값(5일) 기준으로 가장 많이 추천된 종목 Top 10이 순위와 함께 표시된다

성공 기준:
- [ ] 트렌드 랭킹 섹션에 최대 10개의 종목이 순위(1위~10위)와 함께 표시된다
- [ ] 각 종목에 추천 건수가 함께 표시된다
- [ ] 기간 선택 기본값은 5일이다

### 5. 트렌드 랭킹 기간 변경

- **Given** — 트렌드 랭킹 섹션이 5일 기준으로 표시된 상태
- **When** — 사용자가 기간 선택에서 "20일" 또는 "60일"을 선택한다
- **Then** — 해당 기간 기준으로 집계된 순위가 즉시 업데이트된다

성공 기준:
- [ ] 5일/20일/60일 선택 컨트롤이 존재한다
- [ ] 기간 변경 시 랭킹 순위와 추천 건수가 선택한 기간에 맞게 변경된다

### 6. 데이터 없음 상태

- **Given** — 추천종목 데이터가 아직 수집되지 않았거나 스크래핑에 실패한 상태
- **When** — 사용자가 페이지에 접속한다
- **Then** — 빈 목록 대신 "데이터를 불러오는 중입니다" 또는 "오늘의 추천종목이 아직 없습니다" 메시지가 표시된다

성공 기준:
- [ ] 추천종목이 0개일 때 빈 화면이 아닌 안내 메시지가 표시된다
- [ ] 로딩 중 상태와 데이터 없음 상태가 시각적으로 구분된다

## 불변 규칙

**데이터 일관성**: 목록에 표시되는 상승여력은 항상 `(목표가 - 현재가) / 현재가 × 100`으로 계산된 값과 일치한다. 목표가나 현재가 중 하나가 없는 경우 상승여력은 "-"로 표시된다.

**실시간 범위**: 실시간 갱신은 현재가와 상승여력에만 적용된다. 투자의견, 목표가, 증권사 정보는 백그라운드 스크래핑 주기에 따라 갱신된다.

## 의존성

- 한경 컨센서스(또는 동급 무료 소스)에서 추천종목 데이터를 스크래핑할 수 있어야 한다
- 한국 주식 실시간 시세를 제공하는 외부 소스에 접근할 수 있어야 한다
