# 한국 주식 전문가 추천종목 요약 서비스

## 한 줄 요약

**오늘의 전문가 추천종목**을 한 화면에 요약 — 여러 증권사 리포트를 돌아다닐 필요 없이, 매일 아침 30초면 확인.

## 문제 (HMW)

> 여러 증권사에 흩어진 전문가 추천종목 정보를, 매일 아침 30초 안에 한 눈에 파악하고 투자 판단에 활용할 수 있게 모아줄 수 있을까?

## MVP 범위

1. **종목 리스트** — 오늘 추천된 종목 목록 (증권사, 투자의견, 목표가)
2. **종목 요약 카드** — 종목별 현재가, 전문가 컨센서스(평균 목표가, 의견 분포)
3. **트렌드** — 최근 N일간 가장 많이 추천된 종목 랭킹

## Not Doing (MVP 제외)

- 매매 기능 / 포트폴리오 관리
- 리포트 원문 제공 (저작권 이슈)
- 알림/푸시 기능
- 사용자 계정/로그인

## 데이터 소스

| 순위 | 소스 | 데이터 | 비용 |
|---|---|---|---|
| 1 | 한경 컨센서스 스크래핑 | 전 증권사 리포트, 투자의견, 목표가 | 무료 |
| 2 | 네이버 금융 스크래핑 | 종목별 투자의견, 시세 | 무료 |
| 보조 | pykrx / FinanceDataReader | 주가 시세 | 무료 |
| 보조 | DART API | 공시, 재무제표 | 무료 |

## 기술 스택

- **프론트**: Next.js + shadcn UI
- **데이터**: 한경 컨센서스 스크래핑 → API Route에서 처리
- **보조 데이터**: 시세는 한국거래소/네이버 금융

## 숨은 가정

| 범주 | 가정 | 검증 방법 |
|---|---|---|
| Must Be True | 한경 컨센서스가 안정적으로 스크래핑 가능하다 | MVP에서 바로 검증 |
| Should Be True | 사용자가 매일 추천종목 요약을 확인하는 습관이 생긴다 | 본인이 2주간 써보기 |
| Might Be True | 여러 증권사 의견을 집계하면 더 나은 투자 판단이 된다 | 핵심 증명 후 검증 |
